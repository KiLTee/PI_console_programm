None main.change_dict ( any table_inf,
any bdu,
any key )

Заполняет словарь table_inf данными из структуры bdu по ключу key.

Формирует структурированный словарь с метаданными об уязвимости, включая наименование, идентификатор, описание, класс, ПО, версию и другие атрибуты.

Parameters
table_infСловарь (dict), который заполняется данными (выходной параметр).
bduСтруктура данных (dict/list) с исходной информацией об уязвимостях.
keyКлюч (str/int) для доступа к записи об уязвимости в структуре bdu.
Returns
None (функция модифицирует table_inf «на месте»).
See also
Функция используется в handle_kla_vul() для формирования отчётов.

Основная функция программы — запуск GUI и обработка логики приложения.

Инициализирует окно tkinter, создаёт закладки (notebook), размещает элементы управления (labels, entries, buttons) и связывает их с логикой.
